# Playwright Overview
- Developed by Microsoft in 2020.
- Automation testing tool.
- Supports various devices, OS, and browsers (Chromium, Firefox, WebKit).

# Key Features
- Auto-wait: Automatically waits for elements to be ready before performing actions. default(30sec)
- Native Browser Support: Does not require extra code for different browsers.
- Built-in Runner: Includes an integrated test runner.
- Built-in Reporter: Provides native reporting functionality.
- Tracing and Debugging: Advanced tools for troubleshooting tests.
- Built-in Assertions: Integrated validation and verification support.

#Anotations
-we are adding some extra information to our testcase
-Types
    -skip  (inteltionally skip)
    -only  (when only want to run specific testcase)
    -fixme  (when there is a bug or in process)
    -fail
    -describe  (can create multiple testcase inside)
    -slow     (it will take more time to execute liek 90sec)
    -setTimeout


#To Run testcases
    -npx playwright test
    -npx playwright test relative_path   (test specific file only)
    -npx playwright test -g "title"       (test specific testcase only)
    -npx playwright test --project=Chromium (test specific browser only)
    -npx playwright test --project=Chromium -g "title" (test specific browser and testcase only)
    -npx playwright test --project=Chromium -g "title" --headed --debug (test specific browser and testcase only in headed and debug)

#To See the logs and status 
    -npx playwright show-report
    -npx playwright show-report --port 9999 (when high load we can change port)
